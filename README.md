## 🏗️ En qué creo

- **Conceptos > Frameworks**: si no entiendes el problema, la herramienta no te va a salvar
- **Arquitectura importa**: Clean Architecture, Hexagonal, Screaming Architecture — los cimientos definen el edificio
- **Testing no es negociable**: código sin test es código roto que aún no lo sabe
- **Los atajos cuestan caro**: aprender de verdad requiere esfuerzo, y merece la pena
